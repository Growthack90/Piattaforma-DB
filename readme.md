# DB SAL

## Requirement:
- Install Python3
- Install Virtual Environment (VENV)
    * py -3 -m venv .venv
    * .venv\Scripts\activate
- Install PIP
- Install dependencies:
    * pip install setuptools
    * $ pip install flask
    * $ pip install pandas
    * $ pip install numpy
    * $ pip install seaborn
    * $ pip install mysql-connector-python flask-sqlalchemy

- Check if installed correctly:
    * $ pip list

- Run App
    * flask --app hello run
